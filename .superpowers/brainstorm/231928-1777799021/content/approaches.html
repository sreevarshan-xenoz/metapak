Implementation Approaches

How should we integrate the Robustness Suite into arch-tui?

A

Transaction Middleware (Recommended)

Mechanism: A centralized TransactionManager that wraps all actions.

  • Strongest safety guarantees
  • Atomic "Snapshot -> Run -> Notify" flow
  • Easy to add global rollback button
  • Requires refactoring current action logic
B

Action Hooks

Mechanism: Add pre_exec and post_exec hooks to each individual action.

  • Least intrusive to existing code
  • Flexible: turn snapshots on/off per action
  • State management is scattered
  • Harder to maintain consistency
