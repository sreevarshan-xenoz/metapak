1. Snapshot Providers

We'll implement the SnapshotProvider trait for:

  • BtrfsProvider: Uses btrfs subvolume snapshot.
  • TimeshiftProvider: Calls timeshift --create via CLI.
  • LvmProvider: Uses lvcreate -s.
